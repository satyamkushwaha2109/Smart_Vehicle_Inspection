# Smart_Vehicle_Inspection
A real-time vehicle monitoring system that automatically detects vehicles and extracts license plate numbers using YOLO (Deep Learning) and Tesseract OCR, integrated with a Flask web dashboard and MySQL database. Designed for smart surveillance, tolling, traffic management, and security applications.
